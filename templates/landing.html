Documentos y Evidencias

SecureDocs es un portal seguro diseñado específicamente para albergar archivos sensibles, con almacenamiento en Azure Blobs, base de datos preparada para OCI Autonomous y cumplimiento visible de seguridad OWASP.

Encriptación a Nivel de Celda

Los metadatos sensibles y las notas de evidencias son cifradas en reposo utilizando el algoritmo simétrico Fernet (AES-128 en modo CBC) antes de tocar la base de datos.

Integridad de Datos

Cada documento subido genera automáticamente un hash de verificación SHA-256. Esto asegura que la evidencia no ha sufrido alteraciones ni manipulaciones accidentales en el almacenamiento.

Control de Acceso Riguroso

Se mitiga la vulnerabilidad de Broken Access Control (A01:2021) aislando los archivos de cada usuario en consultas separadas y entregándolos mediante URLs firmadas con caducidad corta.
